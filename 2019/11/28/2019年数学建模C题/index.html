2019-11-28  

人在学校，论文已交

首先，感谢两位队友的疯狂输出，毕竟对女孩子来说这种比赛还是很累的（ths)

佛系领队(HHHHH）六点开始比赛，我七点刚刚吃完饭，

拿到题目分析一波


作为一个高中物理可以考个位数的学生，A,B两题实在干不动

A题关于压强
B题受力分析
两道硬核物理实在拿不到

柳暗花明又一村，看到C题，强行分析一波，感觉还不错
第一问：先使用K型聚类，找出影响因素，分析影响机理，再使用层次分析法，得到权重，建立模型
第二问：数据（百度云链接链接: https://pan.baidu.com/s/1_R6Pej4Pbg8afiufF839mg 提取码: tts6 复制这段内容后打开百度网盘手机App，操作更方便哦）


需要进行清洗处理，筛选自己可以使用的数据（费事）
第三问：感谢队友的提醒，否则真的要凉凉，我以为是找时间点或者时间段，题目要求是上车的地点（狗头保命，逃）
深夜分析多排队论模型，又是造轮子的

A、 胡枫. 基于马尔科夫模型的短时交通流预测研究 南京:南京邮电大学,2013.
B、 高雅. 短时交通流预测模型及预测方法的研究 上海:华东师范大学,2011.
C、 张丽. 基于云平台的短时交通流预测算法设计与实现
D、 姚晏斌. 大容量机场轨道交通对陆侧交通的分流预测

再加入特殊影响因素即可

第四问：出租车司机的收益有一个固定值，并且路程和司机收益可以建立函数关系，具体信息可以参考历年数学建模题目出租车配置问题

对于第三问使用原胞自动机模型，进行分析，脑子不够用

关于C题，以前数学建模的时候就已经出过关于出租车的问题，做着并不算太难，毕竟有参考的资料以及文献，

难的是如何做的有新意

不论是第一问 还是第三、四问都很难做到有新意

第三问 我用到的是仿真模拟，当然也可以使用原包自动机模型

可是 这两种方法，不论是实际应用还是历年解法中，都有人使用

如何做到有新意是这一次C题的问题

脑子不够用，想不到有新意的方法，只能在原有的基础上加上自己的理解
