PTA刷题——最大子列和问题

2018-12-03  

7-69 最大子列和问题 （20 分）

给定K个整数组成的序列{N1,N2,...,NK}，“连续子列”被定义为{Ni,Ni+1 , …,Nj}，其中 1≤i≤j≤K。“最大子列和”则被定义为所有连续子列元素的和

中最大者。例如给定序列{ -2, 11, -4, 13, -5, -2 }，其连续子列{ 11, -4, 13 }有最大的和20。现要求你编写程序，计算给定整数序列的最大子列和。

本题旨在测试各种不同的算法在各种数据情况下的表现。各组测试数据特点如下：

数据1：与样例等价，测试基本正确性；
数据2：102个随机整数；
数据3：103个随机整数；
数据4：104个随机整数；
数据5：105个随机整数；

输入格式:

输入第1行给出正整数K (≤100000)；第2行给出K个整数，其间以空格分隔。

输出格式:

在一行中输出最大子列和。如果序列中所有整数皆为负数，则输出0。

输入样例:

6
-2 11 -4 13 -5 -2

输出样例:‘

20

题目分析

还是比较简单的一道题目，只需要每一次读入计算总和就可以了，并且对读入的数字出现相加小于0的现象进行排除就可以了

#include<bits/stdc++.h>
using namespace std;
int main() {
    int i,a,n;
    int sum=0,max=0;
    cin>>n;
    for(i=0; i<n; i++) {
        cin>>a;
        sum+=a;
        if(sum>max) {
            max=sum;
        }
        if(sum<0) {
            sum=0;
        }
    }
    cout<<max<<endl;
    return 0;
}

另一种思路

当然和我的这种思路也是类似的只是使用一个标志

#include<bits/stdc++.h>
using namespace std;
int main() {
    int a[100005];
    int n,sum=0,max=0;
    int f=0;//判断是否输入数字；
    cin>>n;
    for(int i=0; i<n; i++) {
        cin>>a[i];
        f=1;
    }
    if(f=0)
        cout<<0;
    else {
        for(int i=0; i<n; i++) {
            sum+=a[i];
            if(sum<0) {
                sum=0;
            }
            if(sum>max) {
                max=sum;
            }
        }
        cout<<max;
    }
    return 0;
}
